Rainfall
========

Parent of all Rainfall modules.
Rainfall is an extensible java framework to implement custom DSL based stress and performance tests.
See modules :
 - Rainfall-core
 - Rainfall-web
 - Rainfall-ehcache
 - Rainfall-jcache
